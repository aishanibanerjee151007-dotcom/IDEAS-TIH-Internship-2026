# Fake News Detection - IDEAS TIH Summer Internship 2026

## Project
Fake News Detection and Evaluation with Confusion Matrix

## Dataset
The dataset used in this project is publicly available on Kaggle:
https://www.kaggle.com/datasets/emineyetm/fake-news-detection-datasets

Download Fake.csv and True.csv from the above link to run the notebook.

## How to Run
1. Download the dataset from Kaggle link above
2. Upload Fake.csv and True.csv to Google Colab
3. Open the notebook and run all cells in order
